\\\\##Descripción general del proyecto







\\\\\\\*\\\\\\\*Aplicación app\_corazon\\\\\\\*\\\\\\\* es una aplicación  de escritorio diseñada para conectar a los médicos con sus pacientes, facilitando el monitoreo de los signos obtenidos por el tococardiograma.



Características principales



Para el paciente:

&nbsp;	Visualización en tiempo real de los signos vitales (frecuencia cardíaca, ritmo, presión, oxigenación, entre otros).

&nbsp;	Registro histórico de mediciones con posibilidad de graficar tendencias.

&nbsp;	Envío seguro de datos al médico tratante.

&nbsp;	Notificaciones y alertas personalizadas en caso de valores anormales.

Para el médico:

&nbsp;	Acceso a los datos de múltiples pacientes de forma centralizada.

&nbsp;	Herramientas de análisis para la interpretación del tococardiograma.

&nbsp;	Generación de reportes y diagnósticos preliminares.



---





\\\\##  Objetivo y contexto



El objetivo principal del proyecto es desarrollar una aplicación multiplataforma (Flutter Desktop) que permita la interacción fluida entre el usuario y un dispositivo IoT médico, basado en el proyecto OpenCardiographySignalMeasuringDevice. https://github.com/MilosRasic98/OpenCardiographySignalMeasuringDevice



La aplicación integrará la comunicación con el dispositivo mediante conexión Wi-Fi, garantizando la transmisión continua, segura y confiable de datos biomédicos.

Se busca además que la interfaz sea intuitiva y adaptable al rol del usuario (paciente o médico), ofreciendo experiencias diferenciadas y centradas en la accesibilidad.
